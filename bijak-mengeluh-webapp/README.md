# Frontend - Bijak Mengeluh

Next.js 16 web app for complaint generation.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Environment Variables

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=https://brain.bijakmengeluh.id
```

## Deploy

```bash
npm run build
npm run deploy  # or push to main for Vercel auto-deploy
```

See [Deployment Guide](../docs/deployment/deployment-guide.md) for details.
