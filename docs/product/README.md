## Key Decisions (2025-11-16)

**Brand-forward approach:**
1. ✅ Blue for info alerts only (not navigation/buttons)
2. ✅ Orange for all active states (mobile + desktop)
3. ✅ Orange/gray/red buttons only (no blue buttons)
4. ✅ Gray spinners (neutral, not brand)
5. ✅ Info token for alerts only
6. ✅ No color descriptions in copy (use position/label)
